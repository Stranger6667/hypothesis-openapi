Hypothesis plugin for generating valid Open API documents.

## Usage

```python
from hypothesis import given
from hypothesis_openapi import openapis


@given(openapis(version="2.0"))
def test_openapi(spec):
    assert spec["swagger"] == "2.0"
    assert "info" in spec
```
